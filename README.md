# simple-clock-systray
Sets a clock in system tray.
Configurable timezones.

## Requirements
- Go 1.9

## Usage
```
go get github.com/Osuka42g/simple-clock-systray
simple-clock-systray
```

Manual:
```
git clone https://github.com/Osuka42g/simple-clock-systray.git
cd simple-clock-systray
go get
go build
./simple-clock-systray
```
